Hear the Page

Each of the pages on this site contains a Hear It icon in the upper part of the page. Clicking on Hear It will stream the contents of the page for you to listen to as audio output. Each of the pages in this website has been recorded by a human voice in order to make accessible browsing of Ford Mobility Motoring as pleasant as possible. We use Macromedia Flash to bring the content to you.

Keyboard Access Information

We have programmed this site with logical tab controls to help users who have difficulty in using pointing devices such as a mouse. Each page is set up with the option to quickly skip ahead to the navigation area or content section. Additionally, all online forms allow the user to tab through each of the fields in a logical manner.
